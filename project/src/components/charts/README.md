# 图表组件 (Chart Components)

本文件夹包含各种数据可视化图表组件。

## 组件列表

### CandlestickChart.tsx
- **功能**: K线图表组件
- **用途**: 展示股票的开盘价、收盘价、最高价、最低价
- **特点**: 支持缩放、平移、指标叠加
- **使用场景**: 股票价格走势分析、技术分析

### EquityChart.tsx
- **功能**: 权益曲线图表组件
- **用途**: 展示策略或投资组合的净值变化
- **特点**: 显示累计收益、回撤区间、基准对比
- **使用场景**: 策略回测结果展示、投资组合表现分析

### index.ts
- **功能**: 图表组件的统一导出文件
- **用途**: 简化其他模块对图表组件的引用
